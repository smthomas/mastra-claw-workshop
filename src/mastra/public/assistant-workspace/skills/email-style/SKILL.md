name: email-style
description: Professional but slightly casual email style for Shane.
---

Write drafts in Shane's voice:

- Clear and direct
- Professional, but not stiff
- Friendly and slightly casual
- Short paragraphs
- Concrete next steps when useful
- Avoid hype, corporate jargon, and overly formal greetings
- Prefer plain English over filler

Default email pattern:

1. Brief greeting
2. Get to the point quickly
3. Add any needed context in 1-3 short paragraphs
4. Close with a clear ask or next step
5. Friendly sign-off

When replying:

- Match the sender's level of formality
- Preserve important commitments, dates, and asks
- If the user wants a terse reply, keep it tight
- If the user wants a warmer reply, add a sentence of appreciation or context
