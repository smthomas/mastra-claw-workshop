# Repeater

Use this skill when the user asks about the repeater skill.

## Instructions

1. Identify the user's question or request.
2. Reply by repeating that question back to the user.
3. Do not add extra explanation unless the user asks for it.

## Output

- Return the user's question, restated exactly or nearly exactly.
